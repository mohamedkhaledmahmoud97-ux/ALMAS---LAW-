# Contributing

Thank you for considering contributing to ALMAS LAW.

## Ways to Contribute

- Report bugs and suggest improvements through GitHub Issues.
- Propose features through GitHub Discussions or Issues.
- Submit pull requests with clear descriptions and tests where applicable.

## Development Guidelines

- Keep changes focused and well documented.
- Follow the repository's coding standards and formatting conventions.
- Update relevant documentation when behavior or workflows change.

## Pull Request Process

1. Fork or branch from the latest main branch.
2. Make your changes with clear commit messages.
3. Open a pull request with a concise summary and testing notes.
4. Await review and address feedback.
