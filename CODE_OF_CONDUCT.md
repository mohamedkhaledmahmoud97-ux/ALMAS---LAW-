# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, safe, and respectful environment for everyone participating in this project.

## Expected Behavior

- Be respectful and constructive.
- Value different perspectives and experiences.
- Focus on what is best for the community and the project.

## Unacceptable Behavior

Harassment, discrimination, personal attacks, or disruptive behavior are not acceptable.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that do not align with this Code of Conduct.
